# Feedbak project
Crowdsourcing business consulting

## How to start project
- cd into `feedbak` folder
- start frontend on 8080: `npm run start`
- start backend on 9090: `npm run server`

## Project structure
```
feedbak (root)
- server
    - db
    - routes
    - server.js
- src
    - components
    - css
    - App.js
    - index.html
    - index.js

```
